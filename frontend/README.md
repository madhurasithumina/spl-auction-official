# SPL Auction Frontend

React frontend for SPL Cricket Auction System.

## Installation

```bash
npm install
```

## Running the App

Development mode:
```bash
npm start
```

Build for production:
```bash
npm build
```

## Features

- User authentication with local storage
- Player registration form
- Responsive Premier League themed UI
- Protected routes
- Modern animations and styling

## Available Routes

- `/login` - Login page
- `/` - Home page (protected)
- `/register-player` - Player registration (protected)

## Dependencies

- react - UI library
- react-router-dom - Routing
- axios - HTTP client
- react-scripts - Build tools
